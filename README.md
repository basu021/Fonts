## Features
- 70+ fonts for logo design
- Text editing
- Cursive, 3d and signature type fonts
- God of war, Superman + premium fonts
## Installation Commands
1. $ `apt update -y && apt upgrade -y`
2. $ `apt install git -y `
3. $ `git clone https://github.com/basu021/Fonts.git`
4. $ `cd Fonts`
5. $ `chmod +x *`
6. $ `bash run.sh`
